ich kenne 16:9 bzw 16:10 nur von Bekannten, bin aber für 4:3

1.)
für den kurzen Augenabstand finde ich das Blickfeld besser ausgenutzt

2.)
bei gleicher Bilddiagonale beträgt die Fläche für 16:9 nur 89% gegenüber 4:3=100%

3.)
wenn ein 4:3-Bild auf 16:9-Schirm dargestellt wird, verliert mann 33% der Seitenlängen
Wenn ein 16:9-Bild auf 4:3-Schirm dargestellt wird, verliert man nur 8% der Seitenlängen

Fazit: 4:3 für Bildschirmarbeit, 19:9 für Bretwandfilme und so ausgelegte Spiele
