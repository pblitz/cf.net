Dennis.

egal ob VisualBasic, Delphi, C, C++....

JEDE Programmiersprache nutzt zunächst mal die urspünglichste aller Möglichkeiten: Das ist die Kommandozeile, was Du so abfällig als "DOS-Ebene" bezeichnest.

ALLE Sprachen bieten aber auch die Möglichkeit, grafische Oberflächen zu erstellen. Aber das ist eben irre komplex.

Beispiel aus einer anderen Sprache gefällig?

Hier ist "Hallo Welt" in Perl:

Code:
#!/usr/bin/perl

use strict;
warnings;

print "Hallo Welt!\n";


Hier mit grafischer Oberfläche:

Code:
#!/usr/bin/perl

use strict;
warnings;
use Tk;
 
$init_win = new MainWindow;
 
$label = $init_win -> Label(
                            -text => "Hallo Welt!"
                            ) -> pack(
                                      -side => top
                                      );
$button = $init_win -> Button(
                              -text    => "Ok",
                              -command => sub {exit}
                              ) -> pack(
                                        -side => top
                                        );
 
MainLoop;




Und HIER findest Du Hallo-Welt-Beispiele aus den verschiedensten Sprachen - ohne und MIT grafischer Oberfläche.

http://de.wikipedia.org/wiki/Liste_von_Hallo-Welt-Programmen#Perl_mit_Tk

Sehr interessant: Assembler
Code:

.MODEL Small
.STACK 100h
.DATA
  HW      DB      'Hallo Welt!$'
.CODE
 
start:
  MOV AX,@data
  MOV DS,AX
  MOV DX, OFFSET HW
  MOV AH, 09H
  INT 21H
  MOV AH, 4Ch
  INT 21H
end start
