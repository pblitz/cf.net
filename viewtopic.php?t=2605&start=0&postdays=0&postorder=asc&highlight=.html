Du musst nicht unbedingt die Namen deiner Bilder anpassen. Es geht auch wenn du die rot markierten Stellen änderst.

<?php

$zufall = rand(1,5);

switch($zufall)
{
case 1:
$bild = "bild1.jpg";
break;
case 2:
$bild = "bild2.jpg";
break;
case 3:
$bild = "bild3.jpg";
break;
case 4:
$bild = "bild4.jpg";
break;
case 5:
$bild = "bild5.jpg";
break;
}
include("main.html");
?>

Falls du noch mehr Bilder reinhaben möchtest erweitere den "Switch" einfach noch mit einer case mehr:

Code:
    case 6:
    $bild = "bild6.jpg";
    break;


Das kommt dann zwischen das letzte "break;" und die "}" .
