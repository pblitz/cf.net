Also Ubuntu kannst du nicht so leicht in den Bootloader von XP integrieren. Und automatisch schon gar nicht.

Das einfachste wird sein, das du Ubuntu startest, und die Einstellungen des Bootloader von Ubuntu änderst.

Dazu musst du als root die Datei /boot/grub/menu.lst ändern. Darin gibt es einen Default wert, der glaube ich auf 4 geändert werden muss.
Bei der Gelegenheit könntest du auch gleich dem Timeout heruntersetzen.

Nach dem Neustart des Rechners, sollte XP per Default gestartet werden.
