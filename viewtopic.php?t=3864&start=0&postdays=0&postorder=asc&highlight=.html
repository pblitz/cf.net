Hallo zusammen,

ich bin neu hier im Forum und so langsam auch am Ende meiner Nerven. Das surfen ist in den letzten Monaten zur Qual geworden und daher habe ich mir ein paar Gedanken zur Spam-Bekämpfung gemacht.

Folgendes: Manche Webseiten laden einfach nicht fertig, weil in der Statusleiste steht "Warten auf googlesyndication.com" oder "Warten auf doubleclick.net". Ich denke mal dass dies die neumodischen Methoden zur Erfassung von Userstatistiken sind, da ich in meinen Firefox-Einstellungen ebenfalls eine Google-ID gefunden habe. Das ist mir aber recht egal, so lange ich damit keine Probleme bekomme.

Was mir nicht egal ist, dass diese Laderei manchmal bis zu 45 Sekunden dauert und das geht auf die Nerven. Habe mir nun SiteBlock und AdBlock als Plugin heruntergeladen und die bekanntesten Seiten mal außer Betrieb genommen. Das Surfen ist spürbar (etwa doppelt so schnell) schneller geworden.

Meine Frage: Kann ich eventuell auch meinen alten PC zu einem Proxy umrüsten, so dass alle PC's im Netzwerk den Vorteil der blockierten Seiten nutzen können? Außerdem wird es ab 100 Seiten mit der Zeit langsam und belastet unnötig die CPU. Welche Software / Os ist hier empfehlenswert?

Andernfalls - gibt es eventuell Möglichkeiten diese gewünschte Webseiten auch mit der FritzBox! zu sperren?! Habe eine 7050, aber keine Funktion gefunden (vielleicht versteckt per TelNET)?

Jan

Meine erste Wahl wäre in diesem Fall Linux als Betriebssystem und Squid als Proxyserver.

Squid ist eigentlich auf jeder Linux-Distribution enthalten.
Evtl. ist ein vorgebautes Server-Syste wie SME (Linux basiert ) interessant
