Woher weißt du denn was für eine IP der Drucker hat? Vielleicht hat der auch die gleiche IP wie der Router oder einer der PCs...
Lade dir mal SuperScan runter, mit dem kannst du das Netzwerk scannen und sehen welche IPs vergeben sind.
Führe mal so einen Scan durch und checke dann die die Geräte ab, die diese IP haben, so lässt sich einfach ermitteln, welche IP der Drucker hat.

Unter XP musst du im Ordner "Drucker und Faxgeräte" einen Neu hinzufügen und dann "Netzwerkdrucker,..." anklicken. Jetzt musst du die IP im letzten Feld, wo "URL:" davor steht, eingeben.
Ich hoffe ich konnte dir weiterhelfen.
