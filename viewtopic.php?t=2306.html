hihi damit hab ich mich vor kurzem auch rumgeärgert

bei dem mixer problem kann ich dir net helfen. aber vllt einfach mal ohne asoundrc probieren. bzw den ctl-eintrag nochma prüfen

deine kanäle kannst du mit bindingss richtig biegen:so siehts bei mir aus:
Code:

# Wiedergabe device
pcm.playback {
  type dmix               # Wiedergabe Plugin
  ipc_key 1024            # Einzigartiger Key
  ipc_perm 0666           # Rechtezuweisung
  slave.pcm "hw:0"        # Soundkarte
  slave {
    channels 6            # Kanaele
    period_time 0
    period_size 1024
    buffer_size 4096
  }
  bindings {              # Kanal -> Lautsprecher Zuweisung
    0 0                   # 0->0 = Front Left
    1 1                   # 1->1 = Front Right
    2 4                   # 2->4 = Rear Left
    3 5                   # 3->5 = Rear Right
    4 2                   # 4->2 = Center
    5 3                   # 5->3 = LFE
  }

hierzu musst du halt dann dein dmix51 anpassen


PS: ich würd über den LFE nen lowpass jagen, sonst hört sichs erbärmlich an.
