Der Nachteil solcher Spywarescanner ist der, daß sie erst helfen, wenn das Kind schon in den Brunnen gefallen ist.
Keins dieser Tools kann sicher garantieren, daß wirklich jede Spyware gefunden wurde.

Ergo sollte man sich darum bemühen, daß erst gar keine Spyware auf dem Rechner landet, d.h.:
regelmäßig Updates einspielen, Webbrowser entsprechend einrichten (für den Firefox gibt's eine Extension namens Noscript, die so einige Scriptsachen filtern kann), verantwortungsvoller Umgang mit eMail-Anhängen, usw.
