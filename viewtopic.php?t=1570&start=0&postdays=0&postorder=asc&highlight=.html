Habe ein fieses, nerviges, für mich unüberwindbares Problem mit dem Windows Internet Explorer 7 (den ich allerdings unter Windows XP benutze, aber das dürfte ja eigentlich kein Problem darstellen).

Bei nahezu allen Seiten kommt die Meldung eines Skriptfehlers: "In dem Skript auf dieser Seite ist ein Fehler aufgetreten." Er listet in seiner Fehlermeldung irgendwas auf (Zeile, Zeichen, Fehler, Code, URL) und dann stellt er mir die Frage: "Möchten Sie, dass Scripts auf dieser Seite weiterhin ausgeführt werden?" - Ja/Nein.

Egal, ob ich mir für ein "Ja" oder ein "Nein" entscheide, ich kann dann erst einmal weitermachen - bis die nächste Seite geladen werden soll, dann fragt er wieder. Immer wieder. Egal, ob ich das genau bei der selben Seite schon mal entschieden habe!

Vielleicht hängt mit diesem Problem auch zusammen, dass er keine neue Startseite akzeptiert? Er speichert das zwar, wie ich in den Optionen sehe, startet aber weiterhin mit der alten Seite.

Ich habe schon viele Kombinationen unter den erweiterten Internetoptionen ausprobiert, aber nichts funktioniert.

P.S.: Außerdem macht der Explorer beim (automatischen) Aktualisieren einer Internetseite immer blöde Klickgeräusche...
