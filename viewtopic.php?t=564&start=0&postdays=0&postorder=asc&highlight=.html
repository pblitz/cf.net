Zugriffe auf das eigene Netz via DynDNS gehen nicht, auch wenn alles am Router richtig eingestellt ist. Bei HTTP geht es allerdings wenn man im eigenen Browser einen Proxy von aussen benutzt.

Zugriffe von anderen aus dem I-Net funzen aber, es geht nur das eigene interne Netz nicht ohne Proxy. Keine Ahnung warum.

Lange Rede kurzer Sinn versuche mal den FTP Zugriff über einen Proxy von aussen z.B. der deines Providers.
