Einen Hardwarefehler kannst Du ggf. mit memtest prüfen. Dazu musst Du lediglich ein kleines Image auf ein bootfähiges Medium brennen und avon booten - und dann mal über Nacht laufen lassen. Deine Festplatte wird dabei nicht angetastet.

Neuformatiren und installieren hilft bei Windows oft Wunder, würde ich aber erstmal lassen.

Versuche mal herauszufinden was genau sich hinter diesem Prozess verbirgt.
svchost.exe ist ein Prozess der von vielen Diensten/Service ( aber manchmal auch von Malware) gestartet wird. Deswegen kann er durchaus mehrmals im Taskmanager auftauchen.

Anhand des Taskmanagers kann man der ursprünglichen Dienst aber nicht mehr erkennen.

Mit Hilfe des "process explorer" ( http://technet.microsoft.com/de-de/sysinternals/bb896653.aspx ) kannst Du dem auf die Schliche kommen.

Es gibt sogar ein youtube-Video dazu ( http://www.youtube.com/watch?v=bJnEqk4bUjQ )

Du kannst dann gezielt svchost Prozesse beenden und danach versuchen das System herunterzufahren.
