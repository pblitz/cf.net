Hi,

ein BRAS ist ein "Broadband Remote Access Server" und ist sowas wie ein Gateway zwischen DSL-User und IP-Netz des Providers.

siehe auch http://netzikon.net/lexikon/b/bras.html
Zitat:
Der Broadband Remote Access Server (BRAS) ist eine komponente in ADSL-Netzen. Am BRAS terminieren die Sessions der Internetnutzer. Der BRAS ist für Authentification, Authorization und Accounting (AAA) zuständig.
