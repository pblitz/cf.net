so schnell wird der schon nicht "bumm" machen - was wohl lost ist, ist das du ein kabel im Lüfter hängen hast. das hört sich dann so an. Und meistens, wenn du den rechner einschalteste, dreht der den lüfter einmal richtig hoch => Geräusch wird lauter, weil der häufiger an das kabel anschlägt.

Es kann auch sonst einfach sien, das der lüfter lose ist oder sowas, irgendwelche losen Teile im lüfter hören sich so an. Wenn du auf nummer sicher gehen willst: schalten den rechner ein, wenn du ihn auf hast und schau nach, ob sich die lüfter drehen. das ist das wichtigste
