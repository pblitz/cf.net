dispy hat Folgendes geschrieben:


Mit JavaScript scriptet man, mit C/C++/whatever programmiert man.

Dazu müßte man erst definieren was programmieren ist. I.A. versteht man unter Programmieren das implementieren eines Algorithmus in einer Programmiersprache.
Eine Programmiersprache iste eine formale Sprache, mit der man zu jedem berechenbaren Problem (theoretisch) einen Algorithmus implementieren kann, der das Problem löst (Turing-vollständigkeit, so geschrieben, nach Alan Turing, dem Erfinder der Turing-Maschine und damit einem der Väter der Informatik).
Ob diese Sprache schließlich interpretiert, compiliert, zwischenübersetzt oder sonstwas wird, ist letztendlich egal.

Spontan würde ich davon ausgehen, dass JavaScript Turingvollständig ist (d.h. man kann jedes Problem lösen, was man mit einer Turingmaschine lösen kann).

Da man mit HTML nicht rechnen kann ist HTML allein deshalb schon keine Programmiersprache.

Wenn wir schon am Klugscheissen sind
