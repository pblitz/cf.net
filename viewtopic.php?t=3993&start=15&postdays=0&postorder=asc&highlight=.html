Ich habe nun mal einen Testscript mit GD geschrieben, und kann deine Angaben ganz und garnicht nachvollziehen.

Folgender script läuft bei mir mit 0,12 Sekunden durch. Er erzeugt eine Map fast exakt so wei du es in deinem Beispiel genannt hattest (twmaps.org)

Code:
<?php
function microtime_float()
{
    list($usec, $sec) = explode(" ", microtime());
    return ((float)$usec + (float)$sec);
}

$start = microtime_float();

header("Content-type: image/png");
$im     = imagecreate(1000,1000);

$bgCol = imagecolorallocate($im, 70, 120, 40);
imagefill($im, 0, 0, $bgCol);


$col = imagecolorallocate($im, 140, 0, 0);

for ($i = 0; $i < 100000; $i++)
{
  // Variante 1: Pixel
  //imagesetpixel($im, rand(0,999), rand(0,999), $col);

  // Variante 2: Rechtecke
  $x = rand(0, 333)*3;
  $y = rand(0, 333)*3;
  imagefilledrectangle($im, $x, $y, $x+1, $y+1, $col);
}


$lineColor = imagecolorallocate($im, 55,105,33);
for ($x = 13; $x < 1000; $x += 14)
  imagefilledrectangle($im, $x, 0, $x, 1000, $lineColor);....
for ($y = 13; $y < 1000; $y += 14).
  imagefilledrectangle($im, 0, $y, 1000, $y, $lineColor);....

$lineColor = imagecolorallocate($im, 33, 75, 24);
for ($x = 143; $x < 1000; $x += 144)
  imagefilledrectangle($im, $x, 0, $x, 1000, $lineColor);....
for ($y = 143; $y < 1000; $y += 144).
  imagefilledrectangle($im, 0, $y, 1000, $y, $lineColor);....


$end = microtime_float();

$textCol = imagecolorallocate($im, 20, 21, 60);
imagestring($im, 3, 100, 100, "Time: ".($end-$start), $textCol);

imagepng($im);
imagedestroy($im);

?>

Das sollte kein Problem sein, solche Daten verarbeitest du einfach vor, dann ist das einlesen ein klacks. Eine Möglichkeit wäre eine Bit-Karte, also in der FOrm

..XX..
....X..

Die du einfach direkt einliest und bei jedem X auch an der Stelle ein Punkt zeichnest. Das spart Speicher, da du nicht die Korrdinaten Speicherst (20,20), sondern die Datenreihenfolge selbst die Info enthält. Sofern solche einfachen Daten natürlich reichen.

Mit solchen Methoden wurden auch in alten VIdeospielen die Maps gestaltet.
