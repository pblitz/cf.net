es scheint, der PC bootet nach dem ersten Neustart nicht von Platte sondern von CD.

Versuch: für den 1. Neustart die CD rausnehmen und wenn Windows (hoffentlich) beim Starten von der Festplatte ist, wieder einlegen.

Was tut sich dann?

Nachtrag: folgendes könnte die Ursache sein:

-eine Installations-CD XP mit SP2 hat keine Treiber für SATA-Platten
-wenn eine SATA-Platte vorhanden und im BIOS als solche aktiviert ist, wird sie von Windows nicht gefunden, deshalb jedesmal Start von CD

es gibt dann 3 Möglichkeiten:

1. die Treiber auf Diskette mit F6 bei der Installation einbinden

2. die Platte im BIOS als IDE aktivieren, allerdings verzichtet man dann auf die Vorteile von SATA

3. das SP3 in die Installations-CD integrieren, z. B. mit nlite ( www.nliteos.com ). In der Zeitschrift PC-Welt 02/09 gibt es dazu einen Beitrag

2. Nachtrag:
irgendetwas stimmt nicht, denn wohin werden dann bisher die Dateien kopiert? Oder geht das Kopieren, nur das Starten nicht? So ein Experte bin ich nun auch wieder nicht
