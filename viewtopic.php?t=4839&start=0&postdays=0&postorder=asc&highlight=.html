Also bei DVI muß man unterscheiden zwischen

  • DVI-I: Analoge oder Digitales Signal
  • DVI-A: Analoges Signal
  • DVI-D: Digitales Signal

VGA dagegen ist ein analoges Signal. Daher kann es nur einen Adapter DVI-A <-> VGA geben.

DVI-D <-> VGA geht ohne weitere "Intelligenz" per reinem Steckadapter nicht.
