Ich hab dein Problem zwar nicht 100% verstanden, aber kannst du die Daten aus der DB nicht vorverarbeiten?

Sprich mehrfach verschachtelte Arrays erzeugen?
Dann kannst du die Anzahl der enthaltenen Zeilen durch einfaches Count einen Rowspan bzw. Colspan setzen.

In etwa folgende Datenstruktur:

array[spalte1] = array();
array[spalte1][spalte2_wer1] = array()
array[spalte1][spalte2_wert2] = array()
usw.
