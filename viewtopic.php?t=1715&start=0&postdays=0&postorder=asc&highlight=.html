Bekanntermaßen lassen sich bei Kaspersky Internet Security die Port- und Anwendungsregeln sehr selektiv bestimmen, was allerdings einige IT Erfahrung unterstellt.

Die bisher einzig erlaubten Portfunktionen, sind 5198  5200, s. Text unten

sowie DHCP Client Activity UDP beide 67 68 An An
