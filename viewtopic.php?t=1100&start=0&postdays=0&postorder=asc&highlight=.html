ich weiß nicht so genau ob ich hier jetzt richtig bin, denn es betrifft Linux und Windows, aber ich probier es einfach mal.

Also, bisher habe ich vornehmlich mit Windows gearbeitet, zur Zeit XP Sp2. Habe aber auch schon meine Erfahrungen mit Linux gemacht, SuSe und Ubuntu.

Ich habe jetzt schon des öfteren in diversen PC-Zeitschriften gelesen, das es sinvoll sein soll z.B. auf einem Windows Grundsystem eine Virtualisierungssoftware zu installieren (meistens Vmware player), darin eine virtuelle Linux Maschine aufzusetzen, um dann damit ins Internet zu gehen.

Dies sei sicher bzw. sicherer da über die Virtuelle Linuxmaschine keine Viren das Grundsystem angreifen könnten.

Meine Fragen nun:

1: Geht das auch umgekehrt, mit Linux Grundsystem (am liebsten Ubuntu) und darin dann das XP aufsetzen?

2. Wenn ja mit welcher Virtualisierungssoftware am besten?

3. Kann hier das "virtuelle" Windows von außen angegriffen werden oder ist es auch sicher beim surfen?

4. Was ist der Unterschied zwischen Vmware Player und Vmware Server?

5. Kann ich bei einem virtuellen PC mit dem ich ins Internet gehe, diesen Internetzugang auch einem anderen Rechner mit dem ich verbunden bin zur verfügung stellen, also über die virtuelle Maschine routen? Brauche ich dafür vielleicht dieses Vmware server?

1.) ja, natürlich geht das (ist IMHO auch der häufigere Weg für Produktion Systeme)
2.)VMware. Xen oder sowas sind nur Linux auf Linux
3.)Eine Virtuelle Maschine ist ein Computer - mit allen Schwachstellen und Fehlern. Sprich, wenn du mit einer Virtuellen Windows Maschiene in Netz gehst, muss die genauso abgesichert sein wie alles andere. Du kannst allerdings die Hostmaschiene (also das Orginal BS) beim durchrouten einiges machen lassen (stichwort http-proxying, etc)
4.)Player "spielt" schon bestehende Virtuelle Maschienen ab, Server ist ein Server, der mit mehrere Konsolten betrieben werden kann. aber auch nur alleine und auch neue VMs anlegen kann
5.) Das wird wahrscheinlich mit allen VMWare versionen gehen, allerdings würde ich dafür den Server empfehlen. VORSICHT: Das involviert komplexe routing, also würde ich dir davon abraten, wenn du nicht einigermassen firm im normalen Routing bist.
