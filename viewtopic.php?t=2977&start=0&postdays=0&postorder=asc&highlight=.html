Erstmal liegt dein Beitrag im flaschen Forum.

Wenn ich das richtig verstanden habe, willst du von einer 160GB Festplatte ein Sicherungsimage machen. Auf dieser 160GB Platte sind aber nur 30GB Daten drauf, wie du sagst. Dieses Image soll auf eine externe Platte, wo nur 120GB frei sind.

Das Problem bei der Sache sind die freien 120GB auf der externen Platte. Bei einem Sicherungsimage wird nicht nur der belegte Plattenplatz (in deinem Falle 30GB) sondern die gesamte Platte gesichert, also 160GB. (Wie und warum so ein Image funktioniert ist erstmal unerheblich). Fakt ist also, dass du eine 160GB Abbildung auf einer 120GB Platte speichern willst, und das wirklich etwas eng Da hat das Programm schon recht..
