Jetzt ein spaßiger Effekt beim gleichzeitigen Spielen über VPN UND LAN:
Spiel: Revolt
Spielteilnehmer: 1*VPN (Ins Inet), 1*LAN, 1*VPN und LAN (Hoster)
Effekt:
Der Hoster sieht beide Spieler, beide Spieler sehen ihn. Keiner der beiden Spieler kann sich sehen, beide sehen den Host, aber können sich gegenseitig nicht sehen. Der Host sieht beide, kann auf beide schießen. Sie können beide auf ihn schießen (zb. Feuerwerksraketen) und er kann auf sie schießen, aber sie können sich nicht untereinander kabbeln. In der Lobby sehen sie sich auch nicht.
Ergebnis: Kein vernünftiges Spiel möglich
