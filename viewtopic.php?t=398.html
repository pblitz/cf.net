Moin,

denke zur Zeit darüber nach mir eine GPS-Maus zur Navigation zuzulegen. Die Teile gibt es ja schon für 60Eur inkl Navigationssoftware.

Problem:
Mein Notebook (Thinkpad 600E,366MHZ, 128MB RAM) hat nur eine kleine 3,2GB Festplatte auf der ausschliesslich Linux installiert ist. Leider gibt es keine Navi-Software für Linux.

Oder klappt was unter Wine oder einem anderen Emulator?

Ich hätte aber notfalls noch eine 1GB grosse Partition frei, auf der ich Windows + Navi installieren könnte. Zudem noch eine externe Festplatte mit ausreichend Speicherplatz.

Reicht das aus? Auf einer Navisoftwareverpackung habe ich mal gelesen "mind. 3,5GB freier Speicherplatz" als Systemvoraussetzung. Fraglich ist, ob diese Daten sich auch auf eine externe Festplatte auslagern lassen und man sie für die Navigation dann nicht mehr unbedingt braucht.

Gibt es noch eine andere Möglichkeit, als Windows zu installieren? Sonst kann ich Win für nix gebrauchen. Die Software läuft ja auch auf PDAs - wobei dort die Software wahrsch. sparsamer mit Rescourcen umgeht?

Oder ist von der Laptop-Navi generell abzuraten? Brauche die Navi nicht täglich, aber es gab schon massig Situationen, in denen ich gedacht habe - so ein Mist, dass kein Navi an Board ist!

Ach ja: Navigationsoftware soll beinhalten: Deutschland, Niederlande, Frankreich - später dann evtl. erweitert werden auf: Österreich, Schweiz, Spanien. Insbesondere MUSS es die Option geben mautfreie Strecken zu wählen.
Soweit ich weiss, würden die ganzen Strecken eh nicht auf einen billig-PDA passen? Von daher muesste ich wohl sowieso ein Laptop mitschleppen.

Vielen Dank schon mal für eure Hilfe + Tipps!

Gruss
