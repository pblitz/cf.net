Gibts ne Möglichkeit einen Swap nachträglich einzurichten, ohne die bereits bestehende Installation zu verlieren? fdisk sagt zwar "nein" dazu, aber das heißt bekanntermaßen nicht, das es doch gehen würde. fragt sich nur wie ;)
