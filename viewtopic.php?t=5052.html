Ein 16Bit Tastaturtreiber ( Ihr programmiert für DOS? ) unter Windows 7 64 Bit.
Ich bin skeptisch. Die Anleitung bezieht sich sehr wahrscheinlich auf die 32 Bit-Version von Vista.
Du schreibst:
"unser lehrer hat uns ne anleitung gegeben wie es unter windows vista laufen müsste aber das funkt nicht"

Zitat:
... kopieren Sie die Datei keyb2.com in das Verzeichnis C:\TC\BIN

Ebenfalls dorthin kopieren Sie die Datei
C:\Windows\System32\Autoexec.nt
Die Kopie öffnen Sie mit einem Texteditor und fügen am Ende die Zeile
lh C:\TC\BIN\keyb2.com
ein.

... öffnen Sie die Eigenschaften (Explorer/re-Maus) der Datei C:\TC\BIN\tc.exe
wechseln Sie auf die Registerkarte „Programm“,
klicken Sie unten auf „Erweitert“
tragen Sie hinter „Autoexec-Dateiname“ C:\TC\BIN\autoexec.nt ein.


Wo hast Du die Datei keyb2.com hergeholt?
