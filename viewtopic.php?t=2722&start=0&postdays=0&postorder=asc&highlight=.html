Aller Anfang ist schwer - das gilt auch für das Programmieren.
Zum einfachen Zusammenklicken kenne ich nichts.
Dann ist das Blöde an der Programiererei, dass es direkt so viele verschiende Programmiersprachen gibt:

Skriptsprachen wie Javascript, Perl, Php, Python, Visial Basic Script, wo man mit einem normalen Texteditor anfangen kann. Der Programmtext wird von einem Interpreter übersetzt und ausgeführt.
Auf anderen Computern laufen diese Skript-Programme nur, wenn dort ein passender Interpreter installiert ist.

Dann gibt es C, C++, Delphi, Visual Basic - das sind Programmiersprachen, wo man noch einen Compiler braucht, um aus seinem Programmtext ein für den Computer ausführbares Programm zu machen.

Alle diese Sprachen unterscheiden sich in der Schreibweise bei der Programmierung (der "Syntax") mehr oder weniger stark.
Sie unterscheiden sich nicht so sehr, WAS sie ermöglichen, aber sehr darin was sie auf EINFACHE Weise ermöglichen.

Was für Dich interessant sein könnte:

http://www.ciao.de/Visual_Basic_6_fur_Kids_MITP_Weiland_Bernd__236119
oder auch
http://www.perlunity.de/javascript/books/javascript_3826606833.shtml
oder auch
http://www.amazon.de/Game-Programming-Kids-Hans-Georg-Schumann/dp/3826608372/ref=sr_1_10?ie=UTF8&s=gateway&qid=1202112228&sr=8-10

Unter Scherzprogrammen kann ich mir jetzt wenig vorstellen:hast Du mal ein Beispiel?
