Das wird nicht funktionieren wie Du das gerne hättest. Nicht, wenn auf beiden Partitionen Daten sind die Du noch brauchst.
Wenn Du auf Deiner Festplatte 2 Partitionen hast, sagen wir mal C:\ und E:\, und Du möchtest am Ende nur eine grosse Partition C:\ haben, MUSST Du vorher alle Daten von E:\ sichern. Ansonsten sind sie weg.
Dateien werden, wenn mann die Grösse einer Partition verändert, nicht automatisch kopiert oder ähnliches!!

Mache Dir vorher klar WAS auf der anderen Partition drauf ist.

Jetzt habe ich Dich verstanden - allerdings ist die Partition weniger eine Trennwand als eine Schachtel. Es ist auch nicht gesagt ob beide Partitionen das selbe filesystem haben.
=>
1. Sichern
2. Die Daten von der einen Partition auf die andere rüberspielen.
3. Die nun leere Partition löschen
4. Die andere Partition mit dem freien Bereich vergrößern

Das geht nur es hängt von der Art der Daten ab. Hast Du z.B. Daten wie Programme oder Bookmarks, Outlookdateien etc. die Du verschiebst dann findet es im nachhinein diese Dateien nicht mehr weil er sie auf D sucht obwohl sie jetzt auf C sind.

=> Du hast zuerst viel Mühe und dann ein hohes Risiko, dass etwas nicht funktioniert.
Ich weiß nicht was Du Dir davon versprichst aber ich würde die Hände davon lassen!
