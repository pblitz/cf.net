jetzt muss ich mal eine Frage stellen:
Wenn die Logs dieser speziellen IPs so interessant, um nicht zu sagen, euch verdächtig sind, warum setzt Du nicht einen Netzwerksniffer (Nessus, Etherreal o.ä) auf genau diese Datenpakete an und schaust mal nach, was die so transportieren.
Vielleicht gibt das Rückschlüsse, wenn man den Inhalt der Daten mal kennt.
Auch sollte man nicht glauben, dass beim XP-Update nur auf Microsoft-Server und Microsoft IPs zugegriffen wird.
