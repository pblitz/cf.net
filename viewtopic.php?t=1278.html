PHP ist eine Programmiersprache, d.h. Turing vollständig.

In einer Turing vollständigen Sprache kann man alle Aufgaben berechnen, die auf einer Turingmaschine berechenbar sind (also im Prinzip alle Berechnungen - abgesehen vom benötigten Speicherplatz )

Im praktischen Einsatz würde ich sagen, daß der Haupteinsatzzweck von PHP in der Erzeugung dynamischer Webseiten in Verbindung mit SQL-Datenbanken (Mysql, Postgresql,...) liegt. Aberr rein theoretisch kann man auch beliebige andere Progamme in php schreiben. Ich meine sogar gehört zu haben, daß es gtk-Bindings für php gibt, also ein grafisches Benutzerinterface möglich ist.

Man kann so eine Frage nicht beantworten wenn man nicht weiss was du machen willst.

C++ kann dir später nützlich sein, aber es ist für einen Neuling schwer zu erlernen.

PHP ist einfach zu erlernen, und wird dir auch viel im Web nützen, außerhalb ist es ein Exot.

Es gibt Sprachen die vielschichtiger als PHP sind:

Python (die Syntax mag nicht jeder)
Perl (kann fast alles, wegen seiner zu vielfältigen und damit oft unordentlichen Sytnax verschrienen)
Ruby (sehr ordentliche Syntax, leicht zu erlernen, wenig verbreitet)
Java (riesen Monster, sehr beliebt bei Firmen, kann fast alles, läuft auch auf Kaffeeemaschienen )
.Net (Sprachfamilie von MS, nur wirklich sinnvoll für Windows, dort sehr umfangreich und die Zukunft auf Windows)

Dennoch würde ich wegen der Einfachheit im Web bei PHP bleiben, für Desktopanwendungen solltest du dir Java ansehen, wenn es dir garantiert nur um Windows-Desktopanwendungen geht ist .Net mit C# und VisualBasic eine gute Wahl.
