Eine Festplatte ist in Cluster eingeteilt. Die Größe ist vom Betriebssystem abhängig.
512 oder 1024 oder 2048 oder 4096 Byte

Beispiel:
Clustergröße 1024
Es gibt 1000 Dateien.
In jeder Datei steht nur der Buchstabe "A" alse ein Byte.
Der EXPLORER zeigt den Dateiinhalt an (ein Byte).
Der wirkliche Platz ist allerdings
1000 * 1024 = 1024000

Das dauert und wie man sieht ist die Platte mit kleinen Dateien sehr schnell voll.
