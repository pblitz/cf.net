@persitaro wtf = "what the f***"

Was IP-Mapping anbetrifft: jeder versteht etwas anderes darunter, wie es scheint.
Einige benutzen diesen Begriff im Zusammenhang mit IP- und Portumleitung.
Andere bezeichen damit die Abbildung von IP-Adressen auf Geokoordinaten, um eine "IP-Landkarte" aufzubauen.
persitaro bezeichnet damit ständig wiederkehrende Pings, glaube ich.

Eine IP-Adresse ohne DNS Eintrag halte ich nicht für ungewöhnlich.
DNS wurde für den User geschaffen, damit er halt "www.computer-forum.net" schreiben kann statt "85.13.134.193"
In Computernetzen haben aber nicht nur Server eine IP, sondern auch Router, Gateways, Proxies... also nichts, was ein "normaler" User namentlich benennen müsste.
