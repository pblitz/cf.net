Die Helau Gen2Vdr macht folgendes:
Rechner anschalten cd einlegen und warten
Cd wird aus dem Laufwerk geschmissen - du entfernst die CD
CD Lade wird geschlossen und das System merkt das die CD fehlt!
Das ist das Zeichen das kein Monitor am Rechner hängt
CD Lade wird wieder ausgefahren - CD einlegen - Lade schliesst.
Jetzt wird Gen2Vdr auf Platte installiert ohne irgendwelche Fragen es hängt ja kein Monitor dran
Danach zugriff via ssh dazu ist dann aber zumindest netzwerk von nöten.
Dann könnte man sich alles mögliche aus den gentoo repos nachinstallieren.
Samba etc... ist schon alles drauf.
Die Distro ist eigentlch speziell für VDR entwickelt aber was solls man könnte auch nen server draus basteln.
