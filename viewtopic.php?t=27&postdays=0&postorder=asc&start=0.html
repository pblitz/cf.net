Übliches Problem von Windows.

Kann an verschiedenen Dingen liegen.

Beim letzen PC der dieses Problem hatte, war eine BIOS Einstellung schuld daran, dass Win XP so lange brauchte. Am besten ist, wenn man beim CD-Laufwerken im BIOS auf AUTO stellt, dann hat WIndows keine Probleme.

Hängen bleibt er auch oft, wie auch Black Jack sagt, bei Netzwerkzugriffen. Dies kann darin liegen, das die Firewall (wenn eine installiert) hineinfunkt.
