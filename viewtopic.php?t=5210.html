Hi,

mein Betriebssystem ist Windows 7, meine Hardware ist:
Zitat:
CPU: AMD Phenom II X4 965 - 125W - 4x3,4Ghz
GPU: ATI Radeon HD5870 (Sapphire)
RAM: 4x Kingston HyperX 2GB DDR3-1333 - 7-7-7-20
MaB: MSI 790FX-GD70
HDD: 3x Seagate ST31000528AS 1TB
HDD: 2x Samsung HD204UI 2TB
HDD: 2x 160GB, als Hardwareraid 1
DVD: LG GH-22NS
TOW: Thermaltake VH6000BWS Armor+
PSU: Tagan TG500-BZ PipeRock 500W
Kül: EKL Alpenföhn "Nordwand" rev.B
LST: Scythe Kaze Q 3.5"
TFT: 2x HP 2229h 22" - 1680x1050 - 3ms
Mau: Roccat Kone Gaming Mouse (Schwarz)
Pad: Thrustmaster Dual Trigger 3-in-1


Ausgangssituation:
S.o.; 5 Festplatten normal angeklemmt, die beiden 160GB Festplatten als Hardwareraid im Bios als Mirror eingestellt, funktionstüchtig.
SATA Mode war auf IDE eingestellt, Windows 7 wurde unter IDE installiert.

Zielsituation:
SATA Mode soll auf AHCI umgestellt werden um Hot Swap benutzen zu können, in Verbindung mit http://www.idealo.de/preisvergleich/OffersOfProduct/1917998_-sata-quickport-intern-3-bay-sharkoon.html .

Durchgeführte Maßnahme:
1) http://support.microsoft.com/kb/922976
"Windows 7 von IDE auf AHCI umstellen" -> Microsoft "Fix it"-Tool erfolgreich ausgeführt um das System auf die Umstellung vorzubereiten.
2) Umstellung des SATA-Mode in BIOS von IDE auf AHCI

Problem:
Das Festplatten-Raid wird im Windows-Explorer nichtmehr als Laufwerk angezeigt, auch der Gerätemanager und die Datenträgerverwaltung zeigen die Festplatten nicht an. Das Bios erkennt mindestens eine Festplatte beim Starten jedoch und zeigt neuerdings ein neues Bild an. Neben einigen anderen Infos steht dort "Mirror 0 ----------------------------- 159GB" und "Press any key to Continue", eine Fehlermeldung ist nicht erkennbar.

Durchgeführte Maßnahmen zur Problembeseitigung
1) Auf IDE zurückgestellt
2) Daten auf andere Laufwerke gesichert
3) Auf AHCI gestellt und das Raid als Single-Laufwerk eingestellt, d.h. nur eine Festplatte wird benutzt -> Keine Erkennung durch Windows 7
4) Raid physikalisch (kabel) vom PC getrennt, neustart
5) Im BIOS die beiden Laufwerke als Raid 1, Mirror eingestellt -> Keine Erkennung durch Windows 7

Es wurde keine Problembeseitigung erreicht.

Danke - Enomine

Lösung:
1) Treiber-CD für Mainboard (hier MSI 790FX-GD70) einlegen
2) Raid-Treiber installieren
3) neustarten (ACHTUNG DATENVERLUST; Daten wurden in meinem Fall ja bereits gesichert, auf AHCI umgestellt)
4) in der Datenträgerverwaltung den Datenträger neu initialisieren, Laufwerksbuchstaben zuweisen

5) FERTIG
