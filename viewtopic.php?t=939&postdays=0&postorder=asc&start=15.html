Perl würde ich für einen Anfänger nicht wirklich als Alternative zu Python vorschlagen wollen:

Python hat eine sehr klare und strikte[0] Syntax, die man auch versteht, wenn man (noch) kein Python kann. Die Syntax von Perl dagegen gefällt mir nicht unbedingt (Sonderzeichen.....). Zudem kommt der große Umfang von Perlzusatzmodulen hinzu: Es gibt einfach soviele, daß selbst Programmierer die an ähnlichen Codeteilen den Code vom jeweils anderen nicht verstehen [vgl. Kommentar hierzu von esr in TAOUP].


Das "Basissystem" von Python ist auch sehr umfangreich. Vermutlich gibt es nicht ganz so viele Zusatzmodule wie bei Perl, aber auf alle Fälle genügend.

Im Gegensatz zu Perl sind diese leider (noch[???]) nicht an einem zentralen Ort katalogisiert und auch nicht so schön zu installieren, wie bei Perl.
