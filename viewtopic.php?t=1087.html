man fatsort

Code:
       FATSort is a utility that sorts FAT16 and FAT32 partitions. It even can
       handle long file name entries. It reads the boot sector and  sorts  the
       directory structure recursively. This is for example useful for sorting
       the files on your MP3 player when it doesn't implement a sorting  func-
       tion itself.

Zusatz:

Ist das ein mistiger mp3-player.... der hat sogar sien eigenes fat16-System! Mit einem mkfs.msdos formatiert. Player ausgestöpselt.... und schon formartiert der Player sich noch mal selbst.....

Danach dann fleissig kopiert.... und die Fat war zerschossen....

Evtl. kommt fatsort nicht mit dem fat16 des Player klar? Ein passendes Skript habe ich jetzt i.ü. auch gefunden:

Code:
for i in * ; do
      mkdir /mein/mp3/player/"$i" ;
       cp -r "$i"/* /mein/mp3/player/"$i" ;
done


Auch im übrigen macht der Player viel Stress an der USB-Schnittstelle....

Zusatz: Trotz des Skriptes sind eineige Songs doch nicht in der richtigen Reihenfolge! Was solls habe ich eben ein billigen "Shuffle"
