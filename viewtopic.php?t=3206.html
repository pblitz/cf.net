Also:

Die Firma Aplle stellt neben Lifestyleprodukten (ipod, iphone, u.ä.) auch Rechner her.
Ganz früher hatten diese eine 68k-CPU, früher eine PowerPC-CPU (PPC) und aktuell werden intel CPUs verbaut.

Für diese Rechner verkauft Apple acuh ein Betriebssystem, früher war das das klassische Mac OS, jetzt ist es das Mac OS X.

Mac OS X kann nur von Apple bezogen werden und darf nach Lizenzvereinbarung nur auf Rechnern, die von der Firma Aplle hergestellt wurden, installiert werden. Wie bei jedem kommerziellen Betriebsystem (Mac OS X, Windows XP, Windows Vista, AIX, HP-UX, Irix,.....) muß man darauf achten, dass man nur soviele Installationen betreibt, wie man Lizenzen beitzt.
