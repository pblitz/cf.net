Hier ist Code. Der perlige Weg.

Code:
  use Net::ICQ;

  $icq = Net::ICQ->new($uin, $password);
  $icq->connect();

  $icq->add_handler('SRV_SYS_DELIVERED_MESS', \&on_msg);

  $params = {
    'type'         => 1,
    'text'         => 'Hello world',
    'receiver_uin' => 1234
  };
  $icq->send_event('CMD_SEND_MESSAGE', $params);

  $icq->start();


Die vollständige Doku ist hier: http://search.cpan.org/~jmuhlich/Net-ICQ-0.16/lib/Net/ICQ.pm
