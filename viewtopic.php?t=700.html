Ja, so ziemlich jeder Weg geht

Beispiele:

1. Du nutzt PHP und das Kommando "include" um identische Teile in eigene Dateien auszulagern und reinzuladen wenn du sie brauchst.

Code:


<?php

include('header.html');
include('menu.html');

?>

<h1>Toll</h1>

JUP

<?php

include('footer.html');



2. ähnliches geht auch per Javascript, indem du an den stellen wo du immer den gleichen Code willst einfach eine Javascript Datei mit dem inhalt reinlädst. Ist aber nicht so nett als Lösung.

3. Es geht wohl auch direkt mit Apache mit SSI, aber damit habe ich keine Erfahrung.

Frames sollte man heutzutage immer vermeiden, wenn du Frames kann man ein ordentliches Webdesign auvch gleich vergessen.

Am meisten übersicht gewinnt man mit PHP in Verbindung mit einem sauberen STRICT XHTML und CSS Design. Du wirst dich wundern wie klein der Quellcode wird wenn man ihn so schreibt wie es das W3C anregt. So unrecht haben die nicht
