Vermutlich werden die Binaries von Opera und Firefox keine MacOS9-Binaries sein.

Ich glaub kaum, dass es offizielle Binaries von Firefox für alle (exotischen) Plattformen gibt, auf denen er läuft/laufen würde. Wenn man es für eine solche Plattform haben will, dann hilft nur selber bauen (dazu muß man wohl Geduld haben und ausreichend schmerzresistent sein).

Als alternative kann man ältere Browserversionen versuchen. Gerade habe ich gesehen, dass es auf http://www.icab.de/dl.php eine version von iCab3.0.5 gibt, die noch auf PPC/MacOS9 laufen. Wie aktuell die ist, weiß ich nicht.

Ansonsten hab ich von Macs ehrlich gesagt eher wenig Ahnung, hoffe aber dass der icab-Hinweis ein bischen hilft.
