Versuche mal:


Firefox: "Extras" -> "Einstellungen ..." -> Reiter "Anwendungen" -> dort gibt es eine Anwendung "mailto", dort musst du Windows Mail auswählen.

Internet Explorer: "Extras" -> "Internetoptionen" > "Programme" -> dann bei "E-Mail" dein Mailprogramm auswählen.
