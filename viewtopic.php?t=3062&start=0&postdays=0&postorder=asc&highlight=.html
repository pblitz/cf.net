In den Rechner einstecken -> Arbeitsplatz -> Rechtsklick auf den Wechseldatenträger -> Eigenschaften -> bei Dateisystem schauen was dort steht.

Falls dort NTFS steht -> Rechtsklick auf den Wechseldatenträger -> Formatieren -> Bei Dateisystem FAT auswählen.

Beim Formatieren gehen alle Daten verloren!
